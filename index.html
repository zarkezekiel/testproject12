Uploading the files How to install

1 - Unzip the archive downloaded after purchase

2 - Upload "daftplug-instantify.zip" archive with your WordPress admin dashboard in Plugins section

3 - Click on "Activate" button to activate plugin


Usage How to use

1 - Navigate your WordPress admin dashboard

2 - Click on Instantify icon

3 - Choose your prefered settings on each tab and configure options. Every single setting has it's own description about what it does. There is also built-in supoort page with FAQ.
